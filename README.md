# Langkah² install bot dan Module bot
```bash
pkg update && pkg upgrade -y
pkg install nodejs python git -y
git clone https://github.com/CandraMc1909/Vantasy.git
cd mybot
npm init -y
npm install @whiskeysockets/baileys pino qrcode-terminal
```
# Raning Bot
```bash
node index
```
# Add token
```bash
cd Vantasy/database
python craft.py
input panjang token
input banyak token
save ke harian.txt, token.txt, atau vocer.txt
```
